CCS Parking Spot Detection Project
This website was created for CCS Computing 1L by Jared Fitton and Akshay Heda. The goal of this project was to allow faculty and staff of CCS to determine if there are parking spots open behind the CCS building. We would like to thank Professor Conrad for all his help in guiding this project.
This project was built using python, OpenCV, flask, and html/css.
Future ambitions for this project include:
  • Use computer vision to detect amount of available parking spaces
  • Provide push notifications to CCS staff when parking spaces become available
  • Analyze historical data to provide you with the best times to park
  • Spread camera coverage to include all CCS parking lots
How Can You Help?
In order to implement more features we will need to install an outward facing camera that can monitor the half of the parking lot that is closest to the Santa Rosa Resident Hall. Being able to observe the back parking lot from two angles will allow us to have a good enough field of view to properly detect cars using computer vision.
