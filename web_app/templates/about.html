CCS Parking Spot Detection
This website was created for CCS Computing 1L by Jared Fitton and Akshay Heda. The goal of this project was to allow faculty and staff of CCS to determine if there are parking spots open behind the CCS building. We would like to thank Professor Conrad for all his help in guiding this project.
This project was built using python, OpenCV, flask, and html/css. Future ambitions for this project include detection of the amount of cars present using computervision, push notifications and historical data.
